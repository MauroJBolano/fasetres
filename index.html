El objetivo principal del OVI es presentar la estructura básica de HTML5 y la introducción a CSS3, Estructura básica de CSS3, Aplicación de elementos de CSS3 a un proyecto de diseño Web

Objetivo específico 1

Comprender el uso básico de CSS3 para aplicar en el estilo de páginas web.

Objetivo específico 2

Conocer que es CSS3, y como aplicar estilo en la creación de una página web en HTML.
